# AI Case Generator Demo

这是一个使用AI Agent生成：需求文档、功能模块、测试用例、接口文档和压测脚本的演示项目

## 核心流程

1. 创建项目
2. 上传需求文档
3. AI评估需求文档
   - 需求是否合理
   - 需求的风险
   - 需求有哪些不明确的点
   - ...
4. 和用户沟通需求，直至需求清晰明朗
   - 需求文档生成并与用户确认
   - 生成markdown需求文档，可导出
5. 存储需求文档至该项目的向量数据库
6. 根据需求文档拆分系统模块和子模块
   - 与用户确认模块划分
   - 生成markdown模块划分文档，可导出
7. 存储模块划分文档至该项目向量数据库
8. 生成测试用例（用例分级）
   - 与用户确认测试用例
   - 生成Excel测试用例文档，可导出
9. 存储测试用例至该项目向量数据库
10. 根据模块和子模块拆分具体功能接口
    - 与用户确认接口文档
    - 生成markdown接口文档，可导出
    - 生成postman和curl脚本，可导出
11. 存储接口文档至该项目向量数据库
12. 评估关键接口生成locust压测脚本并保存和导出
> 用户可在中途临时保存项目，下次可重新进入已进行的某个阶段继续工作

## 技术栈

- 语言：Python
- 前端框架：Streamlit
- 后端框架：FastAPI、Uvicorn、Pydantic
- AI框架：LangChain、LangGraph
- AI模型：Ollama、Qwen2.5
- 数据库：Chroma、SQLite
